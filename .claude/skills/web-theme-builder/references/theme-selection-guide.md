## 1. Modern/Minimalist

**When to use**: SaaS products, tech startups, design agencies

**Characteristics**:
- Maximum whitespace for breathing room
- Monochromatic colors (blacks/whites/grays)
- Clean sans-serif typography
- Simple geometric shapes
- Subtle shadows and borders
- Minimal decorative elements

**Examples**: Apple, Stripe, Linear, Notion

**Best for**: Products emphasizing simplicity, elegance, professionalism

**Color Palette**:
```
Primary:    0 0% 10%     (near black)
Secondary:  0 0% 30%     (dark gray)
Accent:     0 0% 50%     (medium gray)
Background: 0 0% 100%    (white)
Surface:    0 0% 98%     (off-white)
```

**Typography**: Inter, SF Pro, Helvetica Neue

**Use Cases**:
- Developer tools
- Design software
- Productivity apps
- Premium services

## 2. Corporate/Professional

**When to use**: Enterprise software, financial services, consulting firms, B2B

**Characteristics**:
- Trust-building blues and neutral tones
- Structured, grid-based layouts
- Conservative, readable typography
- High contrast for accessibility
- Professional imagery
- Clear visual hierarchy

**Examples**: IBM, Microsoft, Salesforce, LinkedIn

**Best for**: Establishing credibility, professional authority, enterprise clients

**Color Palette**:
```
Primary:    220 70% 50%  (corporate blue)
Secondary:  220 20% 40%  (slate)
Accent:     200 80% 45%  (bright blue)
Background: 220 20% 98%  (cool white)
Surface:    220 15% 95%  (light gray)
```

**Typography**: IBM Plex Sans, Source Sans Pro, Roboto

**Use Cases**:
- Enterprise SaaS
- Financial dashboards
- Legal services
- Healthcare platforms

## 3. Creative/Bold

**When to use**: Creative agencies, art galleries, entertainment, events

**Characteristics**:
- Vibrant, contrasting colors
- Experimental, asymmetric layouts
- Unique display typography
- High energy visual elements
- Dynamic animations
- Bold imagery and graphics

**Examples**: Awwwards winners, Dribbble showcases, Spotify

**Best for**: Standing out, expressing creativity, appealing to younger audiences

**Color Palette**:
```
Primary:    280 80% 60%  (vibrant purple)
Secondary:  330 85% 55%  (hot pink)
Accent:     45 95% 55%   (golden yellow)
Background: 260 30% 10%  (dark purple)
Surface:    260 25% 15%  (slightly lighter)
```

**Typography**: Clash Display, Space Grotesk, Outfit

**Use Cases**:
- Creative portfolios
- Music/entertainment sites
- Event promotion
- Fashion brands

## 4. Dark/Tech

**When to use**: Developer tools, gaming, tech products, modern software

**Characteristics**:
- Dark backgrounds (true black or near-black)
- Neon/bright accent colors
- Futuristic, technical feel
- High contrast elements
- Monospace elements for code
- Glowing effects and gradients

**Examples**: GitHub Dark, Vercel, Discord, VS Code

**Best for**: Technical products, developer-focused tools, modern aesthetic

**Color Palette**:
```
Primary:    170 80% 50%  (cyan/teal)
Secondary:  280 70% 60%  (purple)
Accent:     45 100% 60%  (amber)
Background: 220 20% 8%   (near black)
Surface:    220 15% 12%  (dark gray)
```

**Typography**: JetBrains Mono (code), Inter (UI)

**Use Cases**:
- Developer tools
- Gaming platforms
- Tech blogs
- API documentation

**Characteristics**:
- Soft, warm color palette
- Rounded corners everywhere
- Inviting, approachable feel
- Friendly illustration style
- Comfortable spacing
- Organic shapes

**Examples**: Airbnb, Etsy, Mailchimp, Headspace

**Best for**: Building trust, warmth, community feeling

**Color Palette**:
```
Primary:    25 90% 55%   (warm orange)
Secondary:  350 70% 60%  (soft coral)
Accent:     45 85% 60%   (sunshine yellow)
Background: 40 30% 98%   (warm white)
Surface:    35 25% 95%   (cream)
```

**Typography**: Nunito, Poppins, Quicksand

**Use Cases**:
- Community apps
- Educational platforms
- Wellness/health apps
- Food & hospitality

## 8. Portfolio/Personal

**When to use**: Personal websites, designer portfolios, freelancers

**Characteristics**:
- Personality-driven design
- Work showcase focus
- Unique, memorable layouts
- Storytelling elements
- Personal brand expression
- Creative navigation

**Examples**: Designer portfolios, personal brands

**Best for**: Individual expression, showcasing creative work

**Color Palette**: Highly variable based on personal brand

**Common Approaches**:
```
Monochrome:     Single hue with variations
Complementary:  Two contrasting colors
Analogous:      Adjacent colors on wheel
Personal:       Brand-specific palette
```

**Typography**: Variable - often display fonts for personality

**Use Cases**:
- Designer portfolios
- Developer portfolios
- Freelancer sites
- Personal blogs

## Combining Themes

Some projects benefit from combining theme elements:

**SaaS + Modern**: Clean SaaS interface with minimalist aesthetics
**E-commerce + Warm**: Friendly shopping experience
**Corporate + Dark**: Professional but modern tech company
**Portfolio + Creative**: Bold personal brand showcase

When combining, choose:
1. **Primary theme**: Sets overall structure and layout
2. **Secondary influence**: Adds accent elements and personality
